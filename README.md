# 📊 Agent Quality Audit Automation

An end-to-end data analytics project designed to analyze customer support interactions and automate agent quality evaluation using Python, SQL, and Power BI.

---

## 🚀 Project Overview

This project simulates a real-world customer support analytics system where raw interaction data is transformed into actionable insights and automated quality metrics.

It combines:
- Data Engineering
- SQL Analytics
- Statistical Analysis
- Business Intelligence
- Quality Audit Automation

---

## 🎯 Project Goals

- Analyze customer support demand and performance
- Identify operational inefficiencies
- Understand drivers of customer satisfaction (CSAT)
- Automate agent quality evaluation
- Build an interactive Power BI dashboard

---

## 🛠️ Tech Stack

- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL (SQLite)**
- **Power BI**
- **Jupyter Notebook**

---

## 📂 Project Workflow

### 1️⃣ Data Modeling
- Converted raw dataset into normalized tables:
  - Customers
  - Orders
  - Agents
  - Interactions

### 2️⃣ Data Ingestion Pipeline
- Built a pipeline with:
  - Logging system
  - Schema validation
  - CSV → SQLite loading

### 3️⃣ Exploratory Data Analysis (EDA)
- SQL-based analysis
- Data validation & cleaning
- Category & workload analysis

### 4️⃣ Feature Engineering
- Created:
  - Resolution Time
  - Handling Time
  - CSAT Categories
  - Customer Remark Flags

### 5️⃣ Summary Table Creation
- Built `customer_support_summary`
- Optimized for analytics & dashboards

### 6️⃣ Statistical Analysis
- Distribution analysis
- Outlier detection
- Correlation analysis

### 7️⃣ Business Insights
- Identified:
  - High-demand categories (Returns, Orders)
  - Long resolution issues (App/Website)
  - Key pain points (Reverse Pickup, Delays)

### 8️⃣ Agent Quality Audit Automation
- Designed rule-based system:
  - Compliance detection
  - Sentiment mapping
  - Script adherence
  - Quality score calculation

### 9️⃣ Power BI Dashboard
- Built interactive dashboard with:
  - KPIs
  - Category insights
  - CSAT analysis
  - Risk detection

---

## 📊 Key Insights

- 📦 Returns & Order issues dominate support demand  
- ⏱️ Resolution time is highly skewed (outliers present)  
- ⭐ CSAT is mostly high (4–5), but not dependent on speed  
- 🚚 Logistics & return processes are major pain points  
- ⚠️ High-risk interactions identified using QA + CSAT
